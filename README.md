# RBAC API

## Overview
role based acess control user management using node, express, mongodb, and accesscontrol. 

## Instructions
Setup the database
> Run:
>
* `docker compose up`

Connect to the database
> Run:
>
* `docker exec -it mongodb bash`
* `mongo -u mongo`

Install
> Run:
>
* `npm install`
* `npm run start`
